LocalLLM 1.1.0 — Stability, Hybrid Engine & UI Refinements
image Image unavailable

Version 1.1.0 focuses on performance optimization, stability fixes, and architectural improvements to the Actalithic engine.

ActalithicCore Hybrid Mode has been refined. The engine intelligently distributes model layers between GPU VRAM and system RAM, allowing larger models to run on mid-range hardware while maintaining acceleration.

Web Worker execution now isolates model computation from the UI thread. This prevents freezing during shader compilation and large model initialization.

Streaming improvements eliminate previous render flicker. Output now appears smooth and continuous without DOM refresh artifacts.

Mobile layout corrections fix padding stacking, picker inconsistencies, and hidden navigation states after model load.

Stability fixes include:

  • Resolved CSS parsing failure caused by stray keyframe fragment
  • Fixed render bubble crash blocking initialization
  • Corrected button visibility state after loadModel()
  • Improved cache handling reliability

LocalLLM 1.1.0 represents the first structural refinement pass after launch — making the system more resilient, responsive, and predictable across devices including Steam Deck, laptops, and hybrid GPU systems.

Feb 18, 2026 Release
LocalLLM 1.0.0 — Public Launch
image Image unavailable

We are officially launching LocalLLM 1.0.0 — a fully browser-native AI assistant that runs entirely on your own hardware. No accounts. No API keys. No servers. Zero data ever leaves your device.

LocalLLM is built on WebGPU acceleration and modern browser compute capabilities. Users can select from multiple model sizes depending on their hardware capabilities — from lightweight 1B models to large high-performance variants.

Launch Highlights:

  • Private, fully local AI execution
  • WebGPU-powered inference
  • Multiple curated model options
  • ActalithicCore hybrid execution mode

This release marks the foundation of the Actalithic platform. Further optimizations and engine improvements follow in subsequent updates.
